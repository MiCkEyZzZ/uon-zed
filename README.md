# uon-zed

Расширение для редактора [Zed](https://zed.dev), добавляющее поддержку формата
данных **УОН** (Универсальная Объектная Нотация / Universal Object Notation).

## УОН — что это?

**УОН** — это русскоязычный формат представления структурированных данных,
вдохновлённый JSON. Ключевые слова и синтаксис ориентированы на русский язык.

### Синтаксис

```uon
{
  имя: "Вася",
  возраст: 25,
  холост: истина,
  дети: пусто,
  дата_рождения: 1999-01-15,
  время_в_пути: 1ч30мин,
  заметка: /* многострочный
               комментарий */,
  // однострочный комментарий
  адрес: {
    город: "Москва",
    индекс: 101000
  },
  телефоны: [
    "+7-900-123-45-67",
    "+7-901-234-56-78"
  ]
}
```

### Поддерживаемые типы данных

| Тип            | Пример               | Описание                  |
| -------------- | -------------------- | ------------------------- |
| `объект`       | `{ ключ: значение }` | Неупорядоченный набор пар |
| `массив`       | `[1, 2, 3]`          | Упорядоченный список      |
| `строка`       | `"текст"`            | Строка в двойных кавычках |
| `число`        | `42`, `3.14`         | Целое или дробное число   |
| `булево`       | `истина`, `ложь`     | Логические значения       |
| `пусто`        | `пусто`              | Отсутствие значения       |
| `дата`         | `2024-01-15`         | Дата в формате YYYY-MM-DD |
| `длительность` | `5д12ч30мин`         | Продолжительность         |

**Ключи** могут быть строками или идентификаторами (в том числе на кириллице).

### Комментарии

- Однострочные: `// комментарий`
- Многострочные: `/* комментарий */`

## Возможности расширения

- ✅ **Подсветка синтаксиса** — грамматика Tree-sitter с подсветкой строк, чисел,
  -дат, длительностей, комментариев, пунктуации
- ✅ **LSP-интеграция** — поддержка языкового сервера `uon-lsp` (должен быть
  установлен отдельно)
- ✅ **Автоматическое определение** — файлы с расширениями `.уон` / `.uon`

## Установка

### Через Zed Extensions

1. Откройте Zed
2. Перейдите в `Extensions` (палитра команд: `extensions: install extension`)
3. Найдите **УОН**
4. Нажмите `Install`

### Вручную

1. Клонируйте репозиторий:
    ```bash
    git clone https://github.com/MiCkEyZzZ/uon-zed.git
    ```
2. Соберите расширение:
    ```bash
    cargo build --release --target wasm32-wasi
    ```
3. Скопируйте полученный `target/wasm32-wasi/release/uon-zed.wasm` в директорию
   расширений Zed.

## Требования

- [Zed](https://zed.dev) (версия с поддержкой WASM-расширений)
- [uon-lsp](https://github.com/MiCkEyZzZ/uon-lsp) — языковой сервер (опционально,
  для LSP-функций)

Установка LSP:

```bash
cargo install --path uon-lsp
```

## Разработка

### Структура проекта

```text
uon-zed/
├── src/lib.rs              # Точка входа расширения (Rust)
├── languages/uon/          # Конфигурация языка
│   ├── config.toml         # Настройки языка
│   └── highlights.scm      # Правила подсветки синтаксиса
├── grammars/uon/           # Tree-sitter грамматика
│   ├── grammar.js          # Описание грамматики
│   ├── src/parser.c        # Сгенерированный парсер
│   └── tree-sitter.json    # Метаданные грамматики
├── extension.toml          # Манифест расширения
└── Cargo.toml              # Rust-зависимости
```

### Сборка

```bash
cargo build --release --target wasm32-wasi
```
